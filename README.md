# pingFun
Having som fun with ping and gtts text-to-speech module

# Rquirements:
* gtts
* pygame

# How to run
1. Clone or download repository
2. Install requirements like so:
```
py -m pip install -r requirements.txt
```
3. Configure the script by running
```
py pingFun.py -c
```
4. Run pingFun.py
```
py pingFun.py
```
